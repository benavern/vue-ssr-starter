# Vue ssr starter

> A ssr oriented way of using vue

## How To

### Dev

```sh
npm run dev
```

it is possible to add `:debug` to debug ssr stuff

### Build

```sh
npm run build
```

and if you want to test the builded version:

```sh
npm start
```
